# 🏠 House Price Prediction (Machine Learning)

## 📌 Overview
This project focuses on predicting house prices using machine learning techniques. It presents a complete workflow including data preprocessing, feature engineering, model training, and evaluation.

The repository is designed as a space for experimenting with different regression models and comparing their performance on housing data.

---

## 🎯 Project Goals
- Build machine learning models to predict house prices
- Prepare and preprocess housing data
- Handle missing values and categorical variables
- Compare different regression approaches
- Evaluate model performance using suitable metrics

---

## 🧠 Technologies Used
- Python
- pandas
- NumPy
- scikit-learn
